# Soil Nutrient Calculator

This tool automates calculation of soil nutrients from laboratory data.

## Features
- Nitrogen (Kjeldahl method)
- Phosphorus (Olsen/Bray)
- Potassium
- Micronutrients (DTPA)

## Inputs
Excel file containing:
- Soil weight
- Extractant volume
- Aliquot volume
- Instrument readings

## Outputs
- Nutrient values in:
  - %
  - mg/kg
  - kg/ha

## How to run
```bash
python main.py
